## 3. HREFLANG IMPLEMENTATION

### Example for Homepage:
```html
<link rel="alternate" hreflang="en-IE" href="https://yoursite.com/en-ie/" />
<link rel="alternate" hreflang="it-IT" href="https://yoursite.com/it-it/" />
<link rel="alternate" hreflang="en" href="https://yoursite.com/en/" />
<link rel="alternate" hreflang="x-default" href="https://yoursite.com/en/" />
```

### Every page must have:
- Self-referencing canonical
- All language variants in hreflang
- Proper x-default
