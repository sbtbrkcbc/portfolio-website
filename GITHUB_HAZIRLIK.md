# 📋 GitHub'a Yükleme Hazırlık Dosyası

## Emergent'te "Save to GitHub" Kullanıyorsanız:

1. Chat ekranında **"Save to GitHub"** butonuna tıklayın
2. Repository adı: `portfolio-website` 
3. Branch: `main`
4. "Push to GitHub" tıklayın
